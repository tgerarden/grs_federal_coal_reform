GRS_data.xlsx contains all inputs for the analysis.

The first tab section contains several tabs with miscellaneous data.

The following tab sections contain the output from the Integrated Planning Model (IPM) that serves as inputs into the analysis in the paper. The paper contains more information on the IPM.

Researchers wishing to conduct analysis using the Integrated Planning Model with alternative assumptions​ beyond those used in this paper may contact ICF International to do so. More information is available at https://www.icf.com/technology/ipm.​​
